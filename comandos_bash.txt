
git init. Cria um repo git na pasta atual
git status 				- Mostra o status do trabalho
git add nome_arquivo - Adiciona arquivo para ser commit
git commit -m "msg" -> Faz commit do que estava para ser
git log - Mostra todo o historico de commit
git checkout nome - troca de branch
git checkout -b nome - cria a branch
git merge branch - faz o merge com a outra branch
git push origin branch - manda pra nuvem
git remote add origin "link"
##### FLUXO DE TRABALHO
1 criar diretorio
2 iniciar git dentro do diretorio criado (git init.
3 edita ou cria arquivos
4  git status
5 git add <nome arquivo>
6 git status
7 git commit -m "mensagem qualquer"
8 git status
9 volta para o passo 3

IMPORTANTE:Trabalho com o repositorio localmente na branch, sobre pro github e faz um pull request pra dar merge com a main no github
